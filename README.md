# Crystallize Plugins

A monorepo of official plugins for [Crystallize](https://crystallize.com).

See the [plugins documentation](https://crystallize.com/docs/developer/plugins) for the full contract and authoring guide.

## Plugins

| Plugin                               | Description                                                |
| ------------------------------------ | ---------------------------------------------------------- |
| [hello-world](./plugins/hello-world) | Reference plugin showing the minimal server-side contract. |

## Building your own plugin

Install the Crystallize plugin skills for Claude Code:

```sh
npx skills add https://github.com/crystallizeapi/ai --skill plugins
```
